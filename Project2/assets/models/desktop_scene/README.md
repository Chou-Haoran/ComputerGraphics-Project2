# desktop_scene

这里是主桌面静物场景的资源目录和场景预设目录。

## 主要资源

- `typewriter/`：打字机 OBJ、MTL、贴图和材质映射
- `table/`：桌子 OBJ、MTL、贴图和材质映射

## 场景预设

- `typewriter_table_showcase.scene`：当前桌子加打字机展示场景
- `typewriter_material_test.scene`：材质测试场景
- `light_types_demo.scene`：灯光类型演示场景
- `area_light_shapes_demo.scene`：面积灯形状演示

## 建议

如果想做最终展示图，优先从 `typewriter_table_showcase.scene` 开始改。
